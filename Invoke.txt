Invoke-Expression
    Given a raw string to invoke
    e.g.
        $foo = "Chrome"
        Invoke-Expression "Get-Process $foo"

    Note:
        Doesn't support the pipeline or executing scriptblocks

Invoke-Command
    Used to execute script blocks
    Given a string you can create a command using [ScriptBlock]::Create("notepad.exe")
    e.g. 
    1.    Invoke-Command { Get-Process Chrome;Get-Process Notepad }
          Avoid [ScriptBlock]:Create() in favor of Invoke-Expression
    2.    $todo = "Get-Process Chrome;Get-Process Notepad"
          $ex = [scriptblock]::create("iex `"$todo`"")
          invoke-command -scriptblock $ex -ComputerName spunk
